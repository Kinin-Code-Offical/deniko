# System Diagrams

This directory contains Mermaid diagrams illustrating the system architecture and key flows.

## Diagrams

| File | Description |
|------|-------------|
| [c4-context.mmd](./c4-context.mmd) | High-level system context (Users, External Systems). |
| [c4-container.mmd](./c4-container.mmd) | Container diagram showing Web App, API, DB, Storage. |
| [c4-components-app.mmd](./c4-components-app.mmd) | Component diagram for the Frontend (Next.js App Router). |
| [c4-components-backend.mmd](./c4-components-backend.mmd) | Component diagram for the Backend (API & Services). |
| [flow-auth.mmd](./flow-auth.mmd) | Authentication sequence (Login, Session). |
| [flow-storage.mmd](./flow-storage.mmd) | File upload and secure retrieval flow. |
| [flow-privacy.mmd](./flow-privacy.mmd) | Privacy enforcement logic for user profiles. |
| [flow-rate-limit.mmd](./flow-rate-limit.mmd) | Rate limiting logic for API protection. |
| [flow-seo.mmd](./flow-seo.mmd) | SEO metadata and sitemap generation flow. |

## Viewing

GitHub renders `.mmd` files natively. You can also use the [Mermaid Live Editor](https://mermaid.live/).
